Overview

Overview

Base UI is a library of unstyled React components and hooks. It provides a robust foundation for creating your own custom design system.

Introduction

Base UI is designed to be fully accessible and customizable. It doesn't ship with any styles, giving you complete freedom to style components as you see fit.

Why use Base UI?

  • Unstyled: No CSS to override. Bring your own styles.
  • Accessible: WAI-ARIA compliant out of the box.
  • Headless: Complete logic and state management without UI constraints.
