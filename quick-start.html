Overview

Quick start

Learn how to get started with Base UI in your React project.

Installation

Base UI is available as an npm package.

npm install @base-ui-components/react

Setup

Base UI components are styled with standard CSS. You can use CSS modules, styled-components, Tailwind CSS, or any other styling solution.

Usage

Import the component you want to use and start building. Here is an example of a simple button component: 

import { Button } from '@base-ui-components/react/button';

export default function App() {
  return <Button>Click me</Button>;
}
